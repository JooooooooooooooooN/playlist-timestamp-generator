# Audio Timestamp Generator

오디오 파일을 순서대로 넣으면 유튜브 챕터용 타임스탬프를 자동으로 생성하는 정적 웹앱입니다.

## 주요 기능

- MP3, WAV, M4A 등 오디오 파일 길이 자동 계산
- 파일 이름을 제목으로 자동 인식
- 제목 직접 수정
- 순서 변경 및 삭제
- 마지막 줄에 `repeat` 같은 문구 자동 추가
- 결과 즉시 복사
- 샘플 플레이리스트 보기
- 파일 업로드 없이 브라우저 안에서만 처리
- 플레이리스트 이름, 전체 메모, 곡별 메모 로컬 저장
- 저장한 플레이리스트 불러오기 및 삭제

## 로컬 사용

1. `index.html`을 브라우저로 엽니다.
2. 오디오 파일을 드래그하거나 `파일 선택`을 누릅니다.
3. 필요하면 제목과 순서를 수정합니다.
4. 필요하면 플레이리스트 이름과 메모를 입력하고 `저장`을 누릅니다.
5. `바로 복사`를 눌러 결과를 사용합니다.

저장한 내용은 사용자의 현재 브라우저 `localStorage`에만 보관됩니다. 서버로 전송되지 않으며, 다른 기기나 다른 브라우저와 자동 동기화되지 않습니다.

이 프로젝트는 서버가 필요 없는 정적 사이트입니다. 아래 같은 정적 호스팅에 바로 올릴 수 있습니다.

- Netlify
- Cloudflare Pages
- Vercel 정적 배포

업로드할 파일:

- `index.html`
- `styles.css`
- `app.js`

## 공개용 운영 팁

- 소개 문구는 영어 또는 한영 혼합으로 유지하면 더 많은 사용자가 바로 이해할 수 있습니다.
- 배포 페이지 설명에 `No upload, processed in your browser` 문구를 함께 적어두면 신뢰도가 올라갑니다.
- 도메인을 붙이면 공유가 훨씬 쉬워집니다.
